# PRD Template Guide

Use `templates/prd.md`.

Keep PRD decisions consistent with:

- policy document rows
- measurement document rows
- AC rows
- `data-req-id`
- decision log entries

Use internal IDs in POLICY, MEASUREMENT, AC, and DECISION_LOG. Keep PRD summaries readable and avoid exposing `POL-*` or `MEA-*` unless the user asks for trace IDs in the PRD.
