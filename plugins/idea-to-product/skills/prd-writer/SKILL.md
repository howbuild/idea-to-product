# PRD Writer

## Role

Write PRD from confirmed decisions.

## Sections

- 요약
- 배경
- 문제 정의
- 목표
- 핵심 사용자
- 사용자 여정
- 핵심 유저 플로우
- MVP 범위
- 제외 범위
- 핵심 기능
- 화면 구성
- 주요 정책 요약
- 측정과 기록 요약
- 성공 지표
- 리스크
- 미결정 사항
- 관련 문서
