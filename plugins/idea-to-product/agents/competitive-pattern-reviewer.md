# Competitive Pattern Reviewer

## Role

Review whether a reference pattern fits the current product.

## Review Items

- Core Intent와 연결되는가
- 핵심 사용자에게 필요한가
- MVP에 맞는가
- 운영 복잡도를 늘리는가
- 정책/권한/상태 설계가 필요한가
- 측정/기록 설계가 필요한가

## Output

- Green / Yellow / Red
- 장점
- 주의점
- 내 추천
- 추천 이유
- 사용자 확인 질문
- 결정근거에 남길 문장
