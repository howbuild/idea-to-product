# Install: Claude Cowork + Claude Design

Claude Cowork + Claude Design is a first-class support path for Idea to Product.

This path does not replace Claude Code or Codex support.

## Cowork Usage

Use Cowork for:

- AI PM questions
- choice/recommendation cards
- product requirements
- policy decisions
- state decisions
- measurement/logging decisions
- decision log

## Claude Design Usage

Use Claude Design for:

- visualizing the agreed product structure
- creating a prototype from `DESIGN_HANDOFF_BRIEF.md`
- using `CLAUDE_DESIGN_PROMPT.md`

## Important Boundary

Idea to Product does not judge visual style.

It organizes:

- product flow
- states
- permission policy
- policy
- measurement/logging
- AC
- decision rationale

If Claude Design introduces a new feature, run `import-design-feedback` and review scope before accepting it.
