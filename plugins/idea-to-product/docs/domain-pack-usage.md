# Domain Pack Usage

## Path-Based Injection

```text
Use domain pack from:
../construction-idea-to-product/domains/construction
```

## Alias-Based Injection

```text
Use domain: construction
```

## Conversation-Based Injection

The user describes the domain in conversation.

## Document-Based Injection

The user provides manuals, policy documents, PRDs, standard forms, or spreadsheets.

## MCP-Based Injection

The runtime reads domain material from Notion, Figma, GitHub, document storage, internal wiki, or similar tools.

## Confirmation Rule

Anything from a domain pack remains a question candidate until the user confirms it.
