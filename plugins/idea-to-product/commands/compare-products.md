# Command: compare-products

Use when several references should be compared for product design decisions.

## Compare By

- 핵심 사용자
- 주요 화면
- 상태값
- 권한정책
- 예외 처리
- 측정/기록
- MVP에 필요한 기능
- v2로 미룰 기능

## Rules

- Do not copy competitors.
- Use comparison only as input for requirements.
- Convert differences into questions, candidates, and decision log entries.
- Distinguish source facts from inference.
