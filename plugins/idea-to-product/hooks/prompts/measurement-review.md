# Measurement Review Prompt

Review whether key user actions, state changes, and operational records are covered.

Check:

- button clicks
- search/filter actions
- create/update/delete/cancel actions
- state changes with who, when, and why
- user flow item: 진입과 다음 행동, 사용 규모, or 품질 기준
- KPI links
- information that should not be recorded
- privacy or sensitive information cautions
